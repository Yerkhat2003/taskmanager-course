# TaskNest

Быстро накиданное приложение для управления досками и задачами на React.

## Что это

Простое веб-приложение где можно создавать доски, добавлять задачи и всё такое. Сделано для демонстрации работы с массивами объектов и динамическим рендерингом.

## Технологии

- React
- Vite
- Tailwind CSS

## Запуск

```bash
npm install
npm run dev
```

## Структура досок

Каждая доска это объект:
- `id` - уникальный номер
- `title` - название
- `description` - описание
- `status` - статус (активная/архивная)
- `createdAt` - дата создания

## Что внутри

- `BoardCard` - компонент карточки доски
- `BoardsList` - список всех досок
- `MainDashboard` - главная страница
- Данные хранятся в state в `App.jsx`
